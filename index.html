creating the iOS 5 folders effect with CSS3 animation, transition and transform.

this example uses jQuery to calculate and set the positions and jquery.cookie.js to save the wallpaper.

optimized as iPhone 4, iOS 5 Homescreen webapp.
This works in -webkit-Browsers only (best in Safari 5 OS X or mobile Safari iOS 5).
Even though I really like Chrome, CSS3-transform animations are really jerky.
